MySQL Database Export & Import Tools

QuickMyImport - Web Interface

Import MySQL database backups with visual progress tracking and resume capability.

  • Upload SQL files (.sql or .sql.zip)
  • Select from existing files in uploads or main directory
  • Automatic .sql.zip extraction (SQL database dumps only)
  • Real-time progress tracking
  • Automatic resume on interruption
  • Session-based processing (no timeouts)
Open Import Tool
Available Files:

The web interface will automatically detect SQL files from:

  • uploads/ directory (uploaded files)
  • Main directory (files in QuickMyDump folder)

Supported formats: .sql (plain SQL) and .sql.zip (ZIP archives containing SQL dumps)
